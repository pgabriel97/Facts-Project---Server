# facts-project
Simple app for managing short memorable facts.

# Environment variables
Have the following environment variables set:
```
# Port on which the server listens for API requests.
SERVER_PORT=<PORT>

# Postgres configuration
PG_HOST=<HOST>
PG_USER=<USER>
PG_PASS=<PASS>
PG_DB=<DB>
PG_PORT=<PORT>
```
